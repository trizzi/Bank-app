Hello folks! This is a Bank app prototype built with Css,React and Node Express. Features include:
Registeration,
Login,
Home,
About.

It was built with functional components using the following hooks:
useState,
useEffect,
useContext,
useReducer,
useRef.

Dependencies for backend include:
Express,
Mongodb,
Mongoose,
JsonWebToken,
Express validator,
uuid,
Bcrypt,
Concurrently,
Nodemon.
